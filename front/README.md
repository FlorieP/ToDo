## Installation
### Front

1. Installer Vue.js:
```bash
  npm install vue
``` 
2. Installer Vuex : 
```bash
  npm install vuex@next
``` 
3. Puis lancez le serveur en tapant la commande : 
```bash
  npm run serve
``` 
- Le serveur doit fonctionner sur [`http://localhost:8080/Login`](http://localhost:8080/Login).
